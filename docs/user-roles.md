# User Roles
## 1. Traveler
A traveler is a user who visits the platform to discover and book homestays.
### Features
* Register and Login
* Search Homestays
* View Property Details
* Book Homestays
* View Booking History
* Leave Reviews
* Get AI Travel Suggestions
---
## 2. Host
A host is a homestay owner who lists and manages properties.
### Features
* Register and Login
* Add New Homestays
* Edit Property Details
* Manage Availability
* View Bookings
* View Ratings and Reviews
* Access Dashboard Analytics
---
## 3. Admin
Admin manages and monitors the entire platform.
### Features
* Manage Users
* Verify Homestay Listings
* Monitor Bookings
* Remove Invalid Content
* Generate Reports
---
